Processing入门<一>

发表于 2016-09-03   |     |  

1. 运行环境的配置

首先从Processing官网下载，下载安装包，参看下图红色箭头，选择no donation出现下载按钮，继续选择进行下载，

然后，根据本机本机类型选择安装包，Windows现在一般都是64位（不放心可以查看一下自己的系统），32位是为了兼容老机器，所以64和32位系统都可以装，Mac OS X只有一种选择，简单。总之下载之后双击就装好了。

装好了如何开始用？不用一脸懵，按照黄色箭头点进去就是Processing的官方教程，第一课就是大神Daniel Shiffman讲的，生动有趣，应该是个不错的开始。建议所有教程都看完，就有了基本操作的理解。

依稀记得当初在安装的时候需要另外安装JRE(Java Runtime Environment)，有的人电脑已经有了就不用理会，如果没有还需要另外安装一下，这一块如果有不清楚的请留言，有需要我再补充。

2. 参考书籍：

1.入门：

爱上Processing：Getting Started with Processing




之前没关注过这本书，上次有同学拿这本书过来问我内容如何，于是回去了解了一下，发现非常适合0基础的入门，可以说简单直接，内容清晰。从很多短小，所见即得的例子入手，在保持轻松有趣的学习路径中，将processing的函数应用，直到基本的循环，对象，数组等编程思想，慢慢渗透进来，而且也有中文版，我想对于完全没有编程经验的同学来说，这本书可算是入门首选。PS：此书由processing语言的创始者所作。

Processing: A Programming Handbook for Visual Designers and Artists(Second Edition)

  • 其中第一版有下载
  • 另外此书已经有中文翻译版，可自行在网上搜索购买：《Processing语言权威指南》

此书也是由processing语言的创始者所作，权威性实至名归，内容翔实丰富，地位相当于牛津英汉词典，分类清晰，也非常方便查阅，是我的入门书，值得推荐。

2.进阶：代码本色：《用编程模拟自然系统》




如果说上一本书教你如何生产构筑自己processing大厦的砖块，这本书就是带你一起盖大楼的。此书作者就是官网教程第一课的老师Daniel Shiffman，水平自不必说，这里他从头到尾讲授了如何用processing模拟一套生态系统，知识点涉及L系统，元细胞自动机，以及初步的神经网络，认真研读将会大有裨益。
