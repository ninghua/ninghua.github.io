4. Hexo主题

NexT这个主题最吸引我的是分类标签，可以根据标签类型自动归类，非常的实用，而且一目了然。



5. Hexo写作：

插入图片

这里之所以把插图的方式单独提出来说，是因为markdown中的图片需要链接在线图片地址，因此直接将图片放在public－>img 文件夹下面引用，会造成正文与图片的分离，极其不方便管理，而且随着文件的增多，几乎是灾难性的。
因此这里我们使用hexo-asset-image来统一图文

  • 首先在 _config.yml 中设置 post_asset_folder:true
  • 然后在Hexo目录执行
    npm install https://github.com/CodeFalling/hexo-asset-image --save
    这样在source文件目录中形成的文件结构是图片包和文件同名在一层的，发布的时候也会自动转换为相关格式
  • 有了这样的目录结构（目录名和文章名一致），以本文为例，以后在只要在Hexo建站教程.md中使用![](Hexo建站教程/next.png) 就可以插入图片了。
